# Getting Started
Install dependencies: <br />
npm i

If pre-commit on husky not working
NPM
npm uninstall husky

npm install -D husky@4

npm install -D husky

YARN
yarn remove husky

yarn add -D husky@4

yarn add -D husky

Solution from: https://stackoverflow.com/questions/50048717/lint-staged-not-running-on-precommit
